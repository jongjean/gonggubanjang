\# 🧭 공구반장 프로젝트 개요 (for VS Code + Copilot)



> 이 문서는 VS Code + GitHub Copilot Chat/Edits 환경에서  

> AI가 프로젝트 맥락을 빠르게 이해하도록 돕는 요약 문서입니다.



---



\## 📦 프로젝트 이름

\*\*공구반장 (Gonggu Banjang)\*\*  

스마트폰 카메라로 공구를 촬영하면  

AI가 자동으로 공구 정보를 인식·정리하는 \*\*무료 공구관리 미니앱\*\*



---



\## 🎯 개발 목표

\- 일반 가정 / 학급 / 소호사무실 / 동아리 등 \*\*100개 미만의 공구\*\* 관리 지원  

\- \*\*100% 무료\*\*, 광고 기반 수익화 가능  

\- \*\*태그·QR·NFC·GPS 불사용\*\*, 오직 \*\*이미지 인식 AI\*\*만 활용  

\- 공구 유지·보수·운영을 효율화하고 데이터로 자산관리 가능하게 함



---



\## ⚙️ 기술 스택



| 구분 | 도구 / 기술 | 설명 |

|------|--------------|------|

| 프런트엔드 | React + Vite + TypeScript | 웹뷰형 UI |

| 백엔드 | Node.js + Express (TS) | API 서버 |

| AI | Google AI Studio (Gemini 2.5 Flash 이미지 계열) | 공구 인식 / JSON 추출 |

| 디자인 시스템 | Toss Design System (TDS) | 레드/블랙/화이트 톤 |

| 개발 툴 | VS Code / Node.js / Android Studio + ADB |  |

| 배포 | 앱인토스 (WebView) | Android/iOS 미니앱 실행용 |



---



\## 🧩 주요 기능



1\. \*\*공구 촬영/업로드\*\* → AI로 이미지 인식  

2\. \*\*자동 메타데이터 생성\*\*  

&nbsp;  - 공구명, 제조사, 모델명, 분류, 스펙, 상태(신품/중고), 구매일, 수명  

3\. \*\*데이터베이스화\*\*  

&nbsp;  - requiredKeys(필수항목), hiddenKeys(숨김항목) 지원  

4\. \*\*대출/반납 기록\*\*  

&nbsp;  - 사용자명, 일시, 상태 로그  

5\. \*\*사건 기록\*\*  

&nbsp;  - 파손/분실/수리  

6\. \*\*AI 신뢰도 표시 및 수정 가능\*\*  

7\. \*\*썸네일 카드 + 플로팅 큰 이미지 뷰\*\*

\## 🎨 UI/UX 컨셉



\- 컬러: \*\*레드(#FF3040) / 블랙 / 화이트\*\*  

\- 버튼: \*\*입체형 사각(3D 버튼)\*\*  

\- 카드: \*\*라운드 + 그림자 강조\*\*  

\- UX: \*\*로딩/에러/재시도 처리\*\* 필수  

\- 화면 흐름: \*\*목록 → 상세(플로팅) → 편집\*\*
