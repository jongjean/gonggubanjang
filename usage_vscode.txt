Copilot / AI에게 줄 기본 요청 예시
✅ 새로운 기능 추가
이 프로젝트는 '공구반장'이야.
React + Express + Google Gemini 기반으로 되어 있어.
TDS 톤(레드/블랙/화이트)을 유지하면서 
Detail 페이지에 '필수/미사용 항목 설정 패널'을 추가해줘.
requiredKeys/hiddenKeys를 체크박스로 편집하고 저장하도록 구현해줘.

✅ 버그 수정
Capture.tsx에서 이미지 업로드 후 두 번째 업로드 때 이전 프리뷰가 남아.
state 초기화와 progress bar 처리를 개선해줘.

✅ 스타일 개선
App.tsx의 카드 목록이 너무 촘촘해.
모바일 1열, 태블릿 2열, 데스크탑 3열 반응형으로 수정하고
버튼과 카드에 동일한 그림자/라운드를 적용해줘.
