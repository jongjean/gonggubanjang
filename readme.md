# 🧰 공구반장 (Gonggu Banjang)

> **스마트폰 카메라 한 번으로 공구를 자동 인식하고 관리하는 AI 미니앱**  
> 일반 가정·학급·소호사무실·동아리 등 **100개 미만의 공구나 장비**를  
> 효율적으로 관리할 수 있도록 설계된 100% 무료 웹뷰 앱입니다.

---

## 🎯 프로젝트 개요

**공구반장(Gonggu Banjang)** 은  
스마트폰 카메라로 공구를 촬영하면 AI가 자동으로 인식하여  
공구명, 제조사, 모델, 스펙, 상태(신품/중고) 등을 데이터베이스화해주는  
**공구 관리 자동화 도우미 앱**입니다.

QR·NFC·GPS 등 보조 태그 없이,  
**이미지 인식만으로 관리가 가능한** 것이 가장 큰 특징입니다.

---

## ⚙️ 주요 기능

| 구분 | 설명 |
|------|------|
| 📸 **AI 인식 등록** | 스마트폰 카메라로 공구 촬영 → AI가 자동 분석 및 정보 생성 |
| 🧾 **메타데이터 관리** | 공구명, 제조사, 모델명, 분류, 스펙, 구매일, 수명, 상태(신품/중고) |
| 🔄 **대출 / 반납 기록** | 사용자명, 일시, 상태 로그 자동 기록 |
| ⚠️ **사건 이력 관리** | 파손 / 분실 / 수리 기록 |
| 🧩 **필수·비필수 항목 관리** | 사용자가 필요한 항목만 선택적으로 관리 |
| 🖼️ **썸네일 + 플로팅 이미지** | 리스트 썸네일 / 상세 플로팅 큰 이미지 제공 |
