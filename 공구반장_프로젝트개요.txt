[프로젝트명]
랜툴박스 (LanToolbox)

[핵심목표]
스마트폰 카메라 기반 공구 관리 미니앱 “공구반장” 개발
- 일반 가정, 학급, 소호사무실, 동아리용 (100개 미만 공구)
- 100% 무료
- 태그/QR/NFC/GPS 없이 이미지 인식 AI로 유지관리 자동화

[기술스택]
- 프론트: Vite + React + TDS 스타일
- 백엔드: Node.js(Express) + Google AI Studio(Gemini 2.5)
- 개발환경: VS Code (또는 Cursor), Node.js, Android Studio, ADB 연결
- 배포: 앱인토스(WebView)

[디자인 컨셉]
- 컬러: 레드 / 블랙 / 화이트
- UI: TDS 톤, 입체 사각 버튼, 단순한 인터랙션
- 썸네일 + 플로팅 큰 이미지 구조

[주요 기능]
1. 스마트폰으로 공구 촬영 → AI로 공구명/제조사/모델 등 자동 인식
2. 기본정보 DB화 (스펙, 수명, 구매일자 등)
3. 대출/반납/사용자 기록 관리
4. 파손·수리·분실 이력
5. 신품/중고 구분
6. 필수데이터·비필수데이터 선택 기능

[정책/보안]
- 서버에 사진 상주 X (임시파일 즉시삭제)
- HTTPS 필수, 테스트만 HTTP 허용
- 외부 로깅 제한
- 개인정보 미수집

[현재 개발상태]
- VS Code 환경 세팅 완료
- Node.js, Android Studio, ADB 연결 완료
- 프런트/백엔드 기본 코드 구조 초안 완성

[차기 단계]
- 필수/미사용 항목 토글 패널 구현
- Cloud Run 배포 자동화
- 앱인토스 콘솔 심사/출시 준비
